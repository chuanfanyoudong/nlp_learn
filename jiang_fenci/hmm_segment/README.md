# 中文分词模型
一个纯粹使用HMM的中文分词模型

# 模型效果

    准确率：0.7416711648494009
    召回率：0.6686293783881109
    F1：0.7032587944456672

# 快速开始
### 模型训练

    python manage.py train

### 模型测试

    python manage.py test
